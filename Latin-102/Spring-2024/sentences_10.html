Latin 102
Active and Passive Periphrastics

1
respondī Clōdiō mē ita esse factūrum.
  • respondī: construction → SBJ responds to OBJ2(Y) that OBJ1(X), where X is an indirect statement and Y is DAT.
  • Clōdiō: MASC SING DAT, PrN of a Roman, Clōdius, Clōdī, m.
  • mē: SBJ of esse factūrum.
  • esse factūrum: = → prospective [time(X)] active, where X is indicated by matrix
2
sed ista rēs nūllō modō relinquenda est.
  • modō: ABL(X) without prep. = "in X"
  • relinquenda est: gerundive + est → debitive present passive
3
cum Antōniō item est agendum, ut cum Cūriōne.
  • cum: prep. + ABL(X) = "with X"
  • Antōniō: MASC SING ABL, PrN of a Roman, Antōnius, Antōnī, m.
  • agendum: = "do";
  • est agendum: gerundive + est → debitive present passive
  • ut: relative adverb: = "as/just as"
  • Cūriōne: MAS SING ABL, PrN of a Roman, Cūriō, Cūriōnis, m.
4
cupiō scīre, quid agās et ubi sīs hiemātūrus.
  • cupiō: construction → SBJ desires OBJ(X), where X is complementary infinitive clause
  • scīre: construction → SBJ knows OBJ(X), where X is indirect question clause
  • quid: interrogative pronoun, neuter singular accusative, = "what?"
  • agās: = "do"; present subjunctive → progressive present
  • ubi: interrogative adverb, = "where?"
  • sīs hiemātūrus: → prospective present active
5
hanc putāvī meā manū scrībendam, itaque fēcī.
  • hanc: SBJ of infin.
  • putāvī: construction → SBJ thinks OBJ(X), where X is indirect statement
  • manū: ABL(X) without prep. can = "by/by means of X"
  • scrībendam: = scrībendam esse → debitive [time(X)] passive, where X is indicated by matrix
  • -que: Xque = "and X"
  • ita: = "thus/so/that"
6
dē frātre, ubi eum vīsūrī essēmus, nesciēbāmus.
  • dē: prep. + ABL(X) = "concerning/about X"
  • ubi: interrogative adverb = "where?"
  • vīsūrī essēmus: prospective past active
  • nesciēbāmus: cnstruction → SBJ does not know OBJ(X), where X is an indirect question
7
scrībis tē nescīre, quid nostrīs faciendum sit.
  • scrībis: construction → SBJ writes OBJ(X), where X is an indirect statement clause
  • tē: SBJ of nescīre.
  • nescīre: construction → SBJ does not know OBJ(X), where X is an indirect question clause
  • quid: interrogative pronoun, neuter single nominative = "what?"
  • nostrīs: DAT(X) can = "through the agency of/by X"
  • faciendum sit: debitive present passive
8
putō enim etiam atque etiam mihi dīcendum esse.
  • putō: construction → SBJ thinks OBJ(X), where X is indirect statement clause
  • etiam atque etiam: = "again and again/over and over"
  • mihi: DAT(X) can = "through the agency of/by X"
  • dīcendum esse: gerundive infinitive → debitive [time(X)] passive, where X is indicated by matrix
9
illud quidem nec faciendum est nec fierī potest.
  • quidem: adverb focusing emphasis on preceding element
  • nec ... nec: = "neither ...nor"
  • faciendum est: gerundive present → debitive present passive
  • fierī: = "be done"
  • potest: construction → SBJ can/is able OBJ(X), where X is a complementary infinitive clause
10
in eōs, quī haec ēgerunt, impetum factūrus vidētur.
  • in: prep. + ACC(X) can = "against X"
  • eōs: see is, ea, id
  • quī: relative adjective
  • haec: neuter plural accusative
  • ēgerunt: = "do"
  • factūrus = factūrus esse → prospective [time(X)] active
  • vidētur: construction → SBJ seems to OBJ(X), where X is indirect statement clause
11
prīdiē Kalendās Sextīlis putō mē Lāodicēae fore.
  • Kalendās: ACC(X) without prep. can = "to/before X"
  • mē: SBJ of fore.
  • putō: construction → SBJ thinks OBJ(X), where X is an indirect statement clause
  • Lāodicēae: FEM SING LOC
  • fore: = futūrum esse → prospective [time(X)] active, where X is indicated by matrix
12
nōn ferunt hominēs, quod vidētur esse tamen ferendum.
  • ferunt: = "bear/endure"
  • hominēs: MASC PLUR NOM
  • quod ... ferendum: autonymous relative clause; it functions as direct object of ferunt.
  • quod: NEU SING NOM, = "what/a thing which"
  • vidētur: cnstruction → SBJ seems to OBJ(X), where X is and indirect object clause
  • esse ... ferendum: gerundive infinitive present → debitive [time(X)] passive, where X is indicated by matrix
13
erat enim quīntō Nōnās ventūrus vesperī, id est hodiē.
  • erat ... ventūrus: → prospective past active
  • quīntō: ABL(X) without prep. can = "at/in/on X"
  • Nōnās: ACC(X) without prep. can = "to/before X"
  • vesperī: adverb, = "in the evening"
14
illī nōndum vēnerant sed erant ventūrī inānēs, imparātī.
  • illī: MASC PLUR NOM
  • erant ventūrī: → prospective past active
  • inānēs: = "empty handed"; secondary predication (the "don't leave the party mad" construction
  • imparātī: secondary predication (the "don't leave the party mad" construction
15
quod dictūrus sum, putō equidem nōn valdē ad rem pertinēre.
  • quod dictūrus sum: autonymous relative clause (clause is "promoted" to function of a noun, just as may happen when adjectives are substantivized); here the function is SBJ of pertinēre.
  • quod: NEU SING ACC
  • putō: construction → SBJ thinks OBJ(X), where X is an indirect statement clause
  • dictūrus sum: prospective present active
  • ad: prep. + ACC(X) = "to X"
  • pertinēre: construction → SBJ pertains to/is relevant to OBJ(X), where X is ad + ACC(Y)
16
ex tuīs litterīs cognōvī Pompeium nostrum in Hispāniam itūrum.
  • ex: prep. + ABL(X) = "from of X"
  • cognōvī: construction → SBJ recognized/learns OBJ(X), where X is indirect statement clause
  • Pompeium: MAS SING ACC, SBJ of itūrum (esse).
  • in: prep. + ACC(X) = "to X"
  • itūrum: = itūrum esse, prospective [time(X)] active, where X is indicated by matrix clause
